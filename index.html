This is a Google Summer of Code organizations details visualizer. [Currently under development.]

How to use:

  1. Select the data to load from drop-down.The bubble chart of all organizations will come up. The size of an organization corresponds to the number of areas they worked on.
  2. Click on any bubble. The corresponding graph and card will come up. (If not in view, please scroll down). The graph is a view of areas associated with the organizations.
  3. To reset, please reload the page

Information taken from Google Summer of Code (GSoC) website. Visualization built using d3 in javascript.
